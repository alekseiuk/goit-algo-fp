# Аналіз ймовірностей сум при киданні двох кубиків  
## Метод Монте-Карло

## Мета роботи
Метою даного завдання є моделювання великої кількості кидків двох гральних кубиків за допомогою методу Монте-Карло, обчислення ймовірностей можливих сум та перевірка правильності отриманих результатів шляхом порівняння з аналітичними (теоретичними) значеннями.

## Аналітичні розрахунки
При киданні двох кубиків існує 36 рівноймовірних комбінацій.  
Кількість комбінацій для кожної суми відома наперед, тому теоретичні ймовірності обчислюються як відношення кількості сприятливих комбінацій до 36.

Наприклад:
- сума 2 → 1/36 ≈ 2.78%
- сума 7 → 6/36 ≈ 16.67%
- сума 12 → 1/36 ≈ 2.78%

## Метод Монте-Карло
У ході роботи було виконано велику кількість імітацій кидків двох кубиків (1 000 000 ітерацій).  
Для кожного кидка визначалася сума очок, після чого підраховувалася частота появи кожної можливої суми.  
Ймовірності обчислювалися як відношення кількості появ певної суми до загальної кількості кидків.

## Порівняння результатів
Отримані за допомогою методу Монте-Карло ймовірності практично збігаються з аналітичними значеннями.  
Невеликі відхилення пояснюються випадковим характером симуляції та зменшуються зі збільшенням кількості ітерацій.

Найбільшу ймовірність має сума 7, що повністю відповідає теоретичним розрахункам.  
Суми 2 та 12 мають найменшу ймовірність, що також підтверджує правильність моделі.

## Висновки
1. Метод Монте-Карло коректно відтворює теоретичний розподіл ймовірностей для кидання двох кубиків.
2. Результати симуляції узгоджуються з аналітичними розрахунками.
3. Зі збільшенням кількості ітерацій точність наближення зростає.
4. Отримані результати підтверджують правильність реалізації алгоритму та розрахунків.
